速さ比べ

２人～４人までのターン制オンライン対戦ゲーム

誰よりも速くゴールを目指せ！

動作環境

Windows

操作方法

  • マウス左クリックで完結
  • サイコロを振る：左クリック
  • メニュー：左クリック

ゲームの始め方

  • このゲームは、2人～4人で遊ぶ対戦ゲームです。まず最初に、誰かが部屋の作成と、パスワードを設定します。ほかの人は、そのパスワードを使って、部屋に参加します。それぞれの始め方は次の通りです。

部屋の作成

  • タイトル画面上をクリックしてください。
  • まず、名前入力をクリック、名前を入力してください。名前を入力し終えたら、決定をクリックしてください。
  • 部屋を作成をクリック、パスワードを入力してください。パスワードを入力し終えたらCreateをクリックしてください。
  • これで部屋を作成することができました！後は一緒に遊びたい人にパスワードを教え、部屋に参加してもらいましょう！
  • 抜けるボタンをクリックすると前の画面に戻ることが出来ます。間違えて部屋を作成してしまった場合やパスワードを変えたい場合に役立ちます。
  • 開始ボタンをクリックするとゲームスタートです！　※「部屋を作った人が、必ず開始ボタンをクリックしてください。他の人がクリックしてもゲームを開始することはできません」

部屋に参加

  • タイトル画面上をクリックしてください。
  • まず、名前入力をクリック、名前を入力してください。名前を入力し終えたら、決定をクリックしてください。
  • 部屋に参加をクリック、教えてもらったパスワードを入力してください。パスワードを入力し終えたらJoinをクリックしてください。
  • これで部屋に参加することができます！

ルール

  • 自分のターンが来たらサイコロを振り、出た目の分だけ進む。
  • 残り時間が０になると、強制的にターンが相手に移ります。
  • プレイヤーの誰かがゴールした時点でゲーム終了！
  • サイコロを振り、誰よりも速くゴールを目指そう！

使用アセット/ライセンス

  • サイコロ Unity Asset Store
  • マップ Unity Asset Store
  • フォント Adobe Fonts
  • Photon PUN2 Photon Engine
